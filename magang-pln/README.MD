## Judul Projek  
Dashboard EAM PT. PLN UP2D Kalselteng  
## Ringkasan Projek  
Dashboard ini dibuat saat program magang untuk membantu dalam memonitoring EAM PT. PLN UP2D Kalseleng dan mendukung dalam mengambil keputusan berbasis data yang _real time_.  
# Tugas  
- Membuat dashboard EAM PT. PLN UP2D Kalselteng yang dapat diakses oleh vendor maupun staff lain yang mudah dipahami.  
- Mempermudah pemantauan kondisi aset secara _real-time_.  
- Mendukung pengambilan keputusan berbasis data.  
## _Tools_  
- Looker Studio  
- Google Sheets  
## Sumber Data  
Data internal perusahaan PT. PLN UP2D Kalselteng.  
## Hasil  
Dashboard memudahkan tim staff memantau aset dan mempercepat proses evaluasi operasional.
